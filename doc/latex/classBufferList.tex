\doxysubsection{详细描述}
A reference-\/counted discontiguous string that can discard bytes from the front 

\begin{DoxyNote}{注解}
Used to model packets that contain multiple sets of headers
\begin{DoxyItemize}
\item a payload. This allows us to prepend headers (e.\+g., to encapsulate a T\+CP payload in a T\+C\+P\+Segment, and then encapsulate the T\+C\+P\+Segment in an I\+Pv4\+Datagram) without copying the payload. 
\end{DoxyItemize}
\end{DoxyNote}
