\doxysubsection{详细描述}
An in-\/order byte stream. 

Bytes are written on the \char`\"{}input\char`\"{} side and read from the \char`\"{}output\char`\"{} side. The byte stream is finite\+: the writer can end the input, and then no more bytes can be written.
